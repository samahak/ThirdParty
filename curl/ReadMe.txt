[Windows]

 - compile.


[Android]
 - export NDK_ROOT=PATH_WHERE_NDK_IS(NDK 경로)
 - sudo spctl --master-disable
 - curl-android-ios 디렉토리로 이동.
 - curl 디렉토리에 curl-7.53.1 안의 파일들을 모두 복사.
 - curl-compile-scripts로 이동하여 ./build-Android.sh 파일 실행.


[iOS]
 - curl-android-ios 디렉토리로 이동.
